Use it to:

## 🔌 Connect the ESP32 to your car

To log data, the ESP32 must talk to the car’s OBD2 port.

Typical setup:
1. Plug the OBD2 adapter into the car.
2. Connect the ESP32 to the adapter.
3. Turn the car key to the accessory or ignition position.
4. Wait for the adapter to power on.
5. Open the logger app on your PC.

If you use a wireless setup:
- Make sure the ESP32 is powered
- Confirm the PC can see the device
- Check that the car ignition is on

## 🛠️ Common setup tips

### App does not open
- Try running it as administrator
- Make sure you extracted the ZIP first
- Check that the file finished downloading

### No data shows up
- Confirm the car ignition is on
- Check the OBD2 connection
- Make sure the ESP32 has power
- Try reconnecting the adapter

### Connection keeps dropping
- Move the ESP32 closer to the adapter or PC
- Use a stable USB cable
- Check power to the board
- Restart the app and reconnect

### Data looks wrong
- Confirm the correct car protocol
- Check that the adapter matches your setup
- Retry the connection after turning the ignition off and on
